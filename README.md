# Counter Application 🔢

A simple React Counter App built using the **useState hook** to manage and update state dynamically.

---

## 🎯 Objective

To understand how state works in React by building a counter with increment, decrement, and reset functionality.

---

## ⚙️ Features

* Increment count
* Decrement count
* Reset count
* Prevent negative values (optional)
* Show message when count reaches zero

---

## 🧠 Concepts Used

* React Functional Components
* useState Hook
* Event Handling
* Conditional Rendering

---

## 📸 Behavior

* Clicking **Increment** ➜ increases count by 1
* Clicking **Decrement** ➜ decreases count by 1
* Clicking **Reset** ➜ resets count to 0
* Displays message: **"Minimum limit reached"** when count is 0

---

## 🛠️ Tech Stack

* React JS
* JavaScript (ES6)
* HTML & CSS
